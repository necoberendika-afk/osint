# User Intelligence Tool - OSINT Platform

A powerful web-based OSINT (Open Source Intelligence) tool that searches for users across gaming and social platforms, performs phone lookups, and provides network information gathering capabilities.

## Features

### 🎮 Username Search Across Platforms
- **Chess.com** - Find player profiles with ratings
- **Lichess** - Get Lichess player information with ratings
- **Minecraft** - Check account availability and UUID
- **Steam** - Search Steam Community
- **Roblox** - Search Roblox for usernames
- **Epic Games** - Find Epic Games profiles

### 📱 Phone Number Lookup
- Validate phone numbers
- Get basic phone information
- Format checking

### 🌐 IP Geolocation
- Get location data (city, region, country)
- Retrieve hostname & organization information
- View coordinates and timezone

### 📋 Domain Information
- DNS A record lookup
- Domain resolution

## How to Use

1. **Open the File**: Double-click `osint.html` to open in your browser
2. **Choose a Tab**: Select the search type you need:
   - Username Search (search multiple platforms)
   - Phone Lookup (get phone information)
   - IP Lookup (geolocation data)
   - Domain Info (DNS records)
3. **Enter Information**: Type the username, phone, IP, or domain
4. **Select Platforms** (if searching usernames - optional)
5. **Click Search**: Results will appear below

## Usage Examples

### Username Search
- Enter: "hikaru" → Find Hikaru Nakamura across chess platforms
- Select platforms you want to search
- Results show if account exists with profile links

### Phone Lookup
- Enter: "+1-555-123-4567" (or various formats)
- Get number validation and format information

### IP Lookup
- Enter: "8.8.8.8" or leave blank for your own IP
- See location, organization, timezone, coordinates

### Domain Lookup
- Enter: "google.com"
- Get DNS A records and resolution info
